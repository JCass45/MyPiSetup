### My Homelab stuff

## My RaspberryPi Setup

Files for bootstrapping my home RaspberryPi setup from a fresh Raspbian image. Everything runs
in a Docker container

### `kingslanding`
`kingslanding` is the media pi. It runs the following services:
* Plex
* NZBGet
* Sonarr
* Radarr
